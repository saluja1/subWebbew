== Description ==
This plugin will add custom made elementor elements to elementor page builder to customize sites with ease.




[Support](https://accesspressthemes.com/support)


== Installation ==
1. Upload `ap-companion` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy !!!
